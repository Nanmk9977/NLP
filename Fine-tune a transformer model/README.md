# 📌 Token Classification using BERT

This project implements a BERT-based Token Classification model for tasks like Named Entity Recognition (NER) and Chunking using the CoNLL-2003 dataset.

## 🚀 Features
- Pretrained BERT (`bert-base-uncased`)
- Tokenization with label alignment
- Dynamic padding using Data Collator
- Evaluation using Precision, Recall, and F1-score
- Inference using Hugging Face pipeline

## 🛠️ Tech Stack
Python, PyTorch, Hugging Face Transformers, Datasets, Seqeval

## 📊 Results
Achieved ~0.91 F1-score with strong precision and recall.


## 🎯 Conclusion
Demonstrates the effectiveness of BERT for sequence labeling with proper preprocessing and evaluation.
