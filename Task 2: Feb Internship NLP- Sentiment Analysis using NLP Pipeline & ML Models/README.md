
# 🎬 Sentiment Analysis using NLP & Machine Learning

## 📌 Overview
This project performs sentiment analysis on movie reviews using Natural Language Processing (NLP) and Machine Learning techniques. The goal is to classify reviews as **positive** or **negative**.

---

## 📂 Dataset
- **Name:** IMDB Dataset  
- **Size:** 50,000 reviews  
- **Classes:** Positive / Negative (balanced)

---

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- NLTK (for NLP preprocessing)
- Scikit-learn (ML models)

---

## 🔄 NLP Pipeline
- Text Lowercasing  
- Removal of URLs & special characters  
- Tokenization  
- Stopword Removal  
- Stemming  

---

## 🧠 Feature Engineering
- Bag of Words (BoW)  
- TF-IDF (used for training)

---

## 🤖 Models Used
- Logistic Regression  
- Naive Bayes  
- Decision Tree  

---

## 📊 Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- F1-Score  

---

## 🏆 Results
- Logistic Regression performed best with highest accuracy  
- TF-IDF outperformed Bag of Words
