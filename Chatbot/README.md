## 🤖 AI Chatbot using Transformers

### 📌 Overview

This project is a simple AI chatbot built using **Hugging Face Transformers** and **PyTorch**. It uses a pre-trained conversational model to generate human-like responses based on user input.

---

### 🚀 Features

* Interactive command-line chatbot
* Context-aware responses
* Uses pre-trained transformer model
* Runs on CPU or GPU

---

### 🛠️ Technologies Used

* Python
* PyTorch
* Hugging Face Transformers

---


### ⚙️ Model Used

* `microsoft/DialoGPT-medium`
